# Controls

| Key | Action |
|---|---|
| M | Open the System |
| R | Mutilation / primary ability |
| V | Dagger Rush / secondary ability |
| G | Shadow Extraction |
| B | Open Shadow screen |
| X | Shadow Exchange |
| Z | Quicksilver |
| C | Ruler's Authority |
| H | Toggle System HUD |
| Left Alt | Dodge using Quicksilver |

All controls can be changed in Minecraft's Controls menu.
